# 穿搭美学理论基础

> 搭调 (DressTune) 的美学理论地基。本文档定义了产品中所有穿搭建议的理论来源。
> 每个技巧的 `why_it_works` 都应该追溯到本文档中的至少一条原则。
>
> 配套结构化数据：`aesthetics_knowledge.json`（供匹配引擎消费）

## 一、比例 (Proportion)

比例是穿搭美学中最基础也最被低估的维度。人眼对比例极其敏感——两个身高相同的人，穿出不同的比例，观感可以相差 10cm。

### 1.1 视觉腰线原理

**核心原则**：腰线位置决定腿身比的视觉起点。

人眼自动将「腰线」（上下装交界处）作为上下半身的分界点。腰线每上移 1cm，腿长视觉增加约 2%。

**穿搭应用**：
- 塞衣是最直接的腰线上移手段。法式半塞上移约 3-5cm，全塞上移约 5-8cm
- 短款上衣（不塞）天然上移腰线
- 高腰下装自带腰线上移
- 腰带的颜色/宽度直接影响腰线被注意的程度

**相关技巧**：`french_tuck`, `full_tuck`, `casual_front_tuck`, `visual_high_waist`

**身形差异**：
- 长躯干 + 短腿用户：腰线上移效果最显著（+2 bonus）
- 腿身比均衡用户：微调即可，无需大幅上移
- 苹果型用户：腰线位置需谨慎——过高暴露腹部问题

### 1.2 分割法则

**核心原则**：横向分割点决定了身体被「读」成几段，以及每段的视觉长度。

**关键分割点**（从全身看）：
| 分割位置 | 视觉比例 | 适用场景 |
|---------|---------|---------|
| 腰线（上衣/下装交界） | 决定上下身比 | 所有人都要关注 |
| 脚踝（裤脚/鞋交界） | 决定腿的终点 | 露踝 = 腿到此结束；盖踝 = 腿延伸至鞋 |
| 手腕（袖口/手交界） | 决定手臂终点 | 露腕 = 手臂利落结束；盖腕 = 邋遢 |
| 领口（脖子/上装交界） | 决定脖子长度 + 脸型 | V领延伸，圆领截断 |

**1/3 vs 1/2 法则**：上 1/3 + 下 2/3 比 1/2 + 1/2 更显高。这就是「上短下长」的理论基础。

**相关技巧**：`cuff_roll`, `sleeve_fold`, `button_rules`, `natural_stack`

### 1.3 纵向延伸技巧

**核心原则**：同色系连续不截断 → 视线纵向流动 → 显高显瘦。

**三种延伸方式**：
1. **同色延伸**：上装与下装同色系 → 从肩到脚一条线（如全黑、全白套装）
2. **鞋裤同色**：鞋与裤子同色 → 腿长延伸到脚底
3. **V 领延伸**：领口 V 型开口 → 脖子向下延伸，脸型修饰

**截断效果**：高对比度的上下装（白上衣 + 黑裤子 + 白鞋）会在腰线和脚踝两处截断 → 三段式 → 显矮。

**相关技巧**：`color_echo`（鞋裤同色变体）, `button_rules`（V领延伸）

### 1.4 横向平衡技巧

**核心原则**：身体最宽处决定整体廓形印象。横向平衡的目标是让视觉重心落在合适的位置。

- 肩宽 > 胯宽（倒三角）→ 需要下身增加量感来平衡
- 胯宽 > 肩宽（梨形）→ 需要上半身增加量感来平衡
- 肩宽 ≈ 胯宽（矩形/沙漏）→ 可以自由调节重心

**相关技巧**：`layering_collar`（增加上半身体积）, `stand_collar`（上半身纵向）

## 二、色彩 (Color)

色彩是穿搭中最容易被注意、却也最容易被做错的维度。

### 2.1 色环基础

**核心工具**：12 色色环

**三种搭配关系**：
1. **邻近色**（色环上相邻 2-3 色）：最安全，几乎不会出错。如蓝 + 绿、橙 + 黄
2. **互补色**（色环上对面）：最大胆，需要控制面积比例。如蓝 + 橙
3. **三角色**（色环上等距三角）：最丰富，需要至少一个颜色「退后」。如红 + 黄 + 蓝

**穿搭中的实际应用**：
- 日常穿搭首选邻近色搭配（90% 不出错）
- 互补色用「大面积中性色 + 小面积互补色」的方式引入
- 初学者建议：全身上下不超过 3 个色相

### 2.2 明度、饱和度、色相 —— 三维色彩

**色相 (Hue)**：什么颜色（红/蓝/绿）

**饱和度 (Saturation)**：颜色的鲜艳程度
- 高饱和 = 抢眼、活跃、难驾驭
- 低饱和 = 安静、高级、易搭配
- 穿搭安全公式：大面积低饱和 + 小面积高饱和

**明度 (Value/Lightness)**：颜色的深浅
- 浅色「轻」，视觉上浮 → 适合放在上半身
- 深色「重」，视觉下沉 → 适合放在下半身
- 上浅下深 = 稳定、显高（符合重力直觉）
- 上深下浅 = 个性、街头（反直觉更有冲击力）

### 2.3 60:30:10 法则

穿搭色彩分配的黄金比例：

| 比例 | 角色 | 应用位置 | 示例 |
|------|------|---------|------|
| 60% | 主色 | 大面积单品（外套/裤子） | 深蓝牛仔裤 |
| 30% | 辅色 | 次大面积（上衣/鞋） | 白色 T 恤 |
| 10% | 点缀色 | 小面积（配饰/袜子/帽子） | 红色帽子 |

**为什么有效**：主色建立基调，辅色提供对比，点缀色给予视觉焦点。三者比例失衡 = 「花哨」（点缀太多）或「单调」（没有点缀）。

**相关技巧**：`color_echo`（在 10% 点缀色中找呼应）

### 2.4 色彩呼应原理

**核心原则**：当两个不同位置出现同一颜色，人眼自动把它们连成一条视觉线。大脑解读为「这个人很会穿」。

**呼应距离与效果**：
- 近距离呼应（鞋-袜 10cm）→ 细节精致感
- 中距离呼应（鞋-腰带 50cm）→ 整体协调感
- 远距离呼应（鞋-帽 150cm）→ 最强视觉流动线

**呼应不是复制**：不必完全相同，同色系（如深蓝+浅蓝）也算呼应。呼应超过 3 处 = 制服感。

**六大呼应模式**：鞋帽呼应、鞋包呼应、袜衣呼应、图案色呼应、金属件呼应、鞋裤同色

**相关技巧**：`color_echo`, `belt_color_echo`

### 2.5 色彩桥接

**核心原则**：当上下身颜色对比强烈时（如红色上衣 + 绿色裤子），用一件中间色单品过渡，降低冲突感。

**桥接策略**：
- 中间色 = 两个对比色的「中间色」（如红→紫→蓝）
- 中间色 = 中性色（白/黑/灰/卡其），万能桥接
- 桥接单品 = 腰带、内搭领口、袜子、围巾等小面积单品

**何时需要桥接**：
- 上下装色环距离 > 4 步（如蓝+橙、红+绿）
- 上下装明度差 > 3 阶
- 你自己感觉「颜色有点冲突」

**相关技巧**：`color_bridging`（进阶篇）

### 2.6 肤色与颜色

**暖皮（黄调）** → 暖色系更和谐：大地色、橙色、黄色、奶油白、橄榄绿
**冷皮（粉调）** → 冷色系更和谐：蓝色、紫色、粉色、纯白、灰
**中性皮** → 均可

**关键**：不是「什么颜色不能穿」，而是「什么颜色更提气色」。颜色越靠近脸部（上衣领口），肤色匹配越重要。

## 三、廓形 (Silhouette)

廓形是穿搭的「第一眼印象」——远看先看到廓形，近看才注意到颜色和细节。

### 3.1 五大基本廓形

| 廓形 | 特征 | 适合身形 | 传递的信号 |
|------|------|---------|-----------|
| **H 型** | 上下同宽，直线轮廓 | 沙漏、梨形 | 利落、现代、中性 |
| **A 型** | 上窄下宽 | 倒三角 | 柔和、优雅、女性化 |
| **Y 型**（T 型） | 上宽下窄 | 梨形 | 力量、权威、男性化 |
| **X 型** | 中间收腰 | 矩形、沙漏 | 经典、性感、曲线 |
| **O 型** | 中间外扩 | 苹果（慎用） | 慵懒、前卫、舒适 |

**穿搭中的应用**：不是改变体型，而是通过穿着方式在现有体型上「造」出想要的廓形。

- H 型身材 + 收腰 = 造出 X 型
- 倒三角 + 宽下装 = 造出 A 型
- 梨形 + 宽肩上装 = 接近 H 型

### 3.2 松紧平衡

**核心原则**：全身上下至少要有一处「收」，否则像穿布袋；至少有一处「松」，否则像穿紧身衣。

**三条黄金规则**：
1. **上宽下窄**：宽松上装 + 修身下装。适合梨形（强调上半身）和想显腿长的人
2. **上窄下宽**：修身/塞衣上装 + 宽松下装。适合倒三角（平衡下身）和街头风格
3. **全宽收腰**：全宽松 + 腰带/塞衣收腰。适合矩形身材制造曲线

**违反规则的情况**：
- 全宽不收腰 = H 型廓形，需要身高和气场（City Boy 风格有意为之）
- 全窄 = 对身材要求极高，日常不推荐

**相关技巧**：`full_tuck`, `french_tuck`, `belt_casual_drape`, `no_belt_styling`

### 3.3 廓形与身形匹配

| 身形 | 推荐廓形 | 为什么 |
|------|---------|--------|
| 梨形 | H 型、Y 型 | 避免 A 型（加宽胯部） |
| 矩形 | X 型（收腰）、H 型 + 层次 | 制造曲线或利用直线优势 |
| 倒三角 | A 型、H 型 | 下身加量感，避免 Y 型 |
| 苹果 | H 型、X 型（V 领收腰） | 避免 O 型 |
| 沙漏 | X 型、H 型 | 强调或保持曲线均可 |

## 四、材质 (Fabric & Texture)

材质是穿搭美学中最被忽略的维度，但它是「质感」和「高级感」的主要来源。

### 4.1 材质光谱

每种材质可以在三个轴上定位：

**硬 ↔ 软**（结构感）
- 硬：丹宁、帆布、皮革 → 结构清晰、利落
- 软：丝绸、羊绒、莫代尔 → 柔顺、流动

**光 ↔ 糙**（光泽度）
- 光：丝绸、漆皮、缎面 → 华丽、正式
- 糙：亚麻、毛呢、做旧皮 → 自然、休闲

**厚 ↔ 薄**（重量感）
- 厚：重磅棉、羊毛呢、羽绒 → 秋冬、稳重
- 薄：丝绸、亚麻、薄棉 → 春夏、轻盈

### 4.2 材质对冲原则

**核心原则**：全身上下用不同材质的对比制造层次感。

**三类对冲**：
1. **硬 + 软**（最经典）：皮夹克 + 丝绸裙、牛仔外套 + 针织内搭
2. **光 + 糙**（高级对冲）：缎面衬衫 + 粗花呢外套、漆皮鞋 + 亚麻裤
3. **厚 + 薄**（季节对冲）：厚毛衣 + 薄纱裙、羽绒服 + 薄 T 恤

**对冲的度**：
- 对冲动 > 2 个层次 → 混乱
- 1 个对冲 → 刚好
- 0 个对冲 → 平淡（但不出错）

### 4.3 材质与季节

| 季节 | 推荐材质 | 避免 |
|------|---------|------|
| 夏季 | 亚麻、轻薄棉、丝绸 | 厚重丹宁、羊毛 |
| 春秋 | 中等磅数棉、轻薄羊毛、丹宁 | — |
| 冬季 | 重磅羊毛、羊绒、羽绒、厚棉 | 亚麻、薄纱 |

**材质过渡感**：换季时（3月/9月）最适合做材质对冲——厚+薄的搭配在这个时段最自然。

## 五、风格 (Style)

风格是穿搭的「语言」——它告诉别人你是谁，或者你今天想成为谁。

### 5.1 六大风格方向

**Clean Fit（精确简约）**
- 核心词：合身、整齐、中性色、质感
- 廓形：微宽松直筒
- 色彩：白/黑/灰/深蓝/卡其
- 材质：纯棉、羊毛、丹宁
- 标志技巧：法式半塞（精确正中对准）、窄边卷裤脚、单解开扣
- 精神内核：less is more，每一厘米都要精确

**City Boy（都市休闲）**
- 核心词：oversized、层叠、街头、随意
- 廓形：廓形宽大
- 色彩：军绿/藏青/卡其/灰/白
- 材质：重磅棉、尼龙、灯芯绒
- 标志技巧：宽松堆叠不塞、宽边双折裤脚、层次叠穿
- 精神内核：故意的宽松，有控制的随意

**韩系轻熟（精致修身）**
- 核心词：修身、精致、柔和、细节
- 廓形：合身微收
- 色彩：燕麦/奶白/雾蓝/浅灰/驼色
- 材质：针织、羊毛混纺、垂感面料
- 标志技巧：规整全塞、窄边卷边配白袜、不对称半塞
- 精神内核：温柔而有力量

**日系简约（自然质朴）**
- 核心词：天然、垂感、层叠、质朴
- 廓形：宽松垂坠
- 色彩：藏青/灰/米白/橄榄绿/棕
- 材质：亚麻、棉麻、牛仔布、毛呢
- 标志技巧：随意前塞、自然堆叠不卷、立领造型
- 精神内核：与衣物对话，穿出时间感

**美式复古（硬朗实用）**
- 核心词：工装、丹宁、做旧、硬朗
- 廓形：合身偏宽松
- 色彩：靛蓝/棕色/橄榄绿/奶油白/铁锈红
- 材质：重磅丹宁、帆布、皮革、羊毛
- 标志技巧：宽边单折卷裤脚、卷袖至肘、厚底靴
- 精神内核：衣服是用来穿的，要能干活

**意式经典（优雅随性）**
- 核心词：优雅、不经意的精致、sprezzatura
- 廓形：合身柔软
- 色彩：海军蓝/驼色/酒红/奶油/橄榄
- 材质：羊毛、丝绸、羊绒、软棉
- 标志技巧：袖口单折（仅解开袖口那颗扣）、衬衫单解开扣、随意垂坠腰带
- 精神内核：看起来毫不费力，实际每处都考虑过

### 5.2 风格混搭原理

**核心原则**：一套穿搭中混入 20% 的「异风格」元素，制造张力和个性。

**80/20 法则**：
- 80% 保持一种风格的完整性
- 20% 用另一种风格的单品或穿法打破

**成功的混搭 vs 失败的混搭**：
- ✅ 韩系轻熟（80%）+ 美式复古工装靴（20%）→ 温柔中有力量
- ✅ Clean Fit（80%）+ 日系立领（20%）→ 干净中有细节
- ❌ 四种风格各 25% → 没有主心骨，杂乱

### 5.3 风格与场合

| 场合 | 推荐风格 | 避免 |
|------|---------|------|
| 上班（普通办公室） | Clean Fit、韩系轻熟 | City Boy（太随意） |
| 上班（创意行业） | 日系简约、意式经典 | — |
| 周末休闲 | City Boy、美式复古 | — |
| 约会 | 韩系轻熟、意式经典 | City Boy（太松垮） |
| 正式活动 | 意式经典 | City Boy、美式复古 |

## 六、身形修饰 (Body Modification)

穿搭的核心能力之一：知道自己的身体特征，然后通过穿着优化视觉印象。

### 6.1 五大身形类型与策略

**梨形（肩窄胯宽）**
- 核心挑战：胯宽容易显腿短、上半身量感不足
- 策略：上移视觉重心（用亮色上装、层次叠穿、宽领口）、明确腰线、选择 A 线少的下装
- 避免：紧身包胯的下装、低腰设计、上深下浅
- 最适合的技巧：`french_tuck`（10 分）、`cuff_roll`（9 分）、`sleeve_fold`（8 分）

**矩形（肩胯同宽，无明显腰线）**
- 核心挑战：直筒轮廓、缺乏立体感
- 策略：制造腰线（塞衣/腰带）、增加上半身层次（叠穿/领口造型）、用颜色打破直线
- 避免：全宽松无收腰、单一颜色从头到脚
- 最适合的技巧：`french_tuck`（9 分）、`color_echo`（8 分）、`layering_collar`（8 分）

**倒三角（肩宽胯窄）**
- 核心挑战：上半身体积感过强、下身显单薄
- 策略：下身增加量感（宽松裤、宽边卷裤脚）、柔化肩线（V 领、解扣）、下身用亮色
- 避免：肩部装饰/垫肩、紧身下装、上浅下深
- 最适合的技巧：`button_rules`（8 分）、`cuff_roll`（8 分）、`full_tuck`（8 分）

**苹果（中段突出，四肢纤细）**
- 核心挑战：腰腹是视觉焦点、容易显臃肿
- 策略：V 领纵向延伸、松量控制（不过紧不过宽）、视线引导至四肢/脸部
- 避免：紧身中段、高腰全塞、横向条纹
- 最适合的技巧：`button_rules`（7 分）、`cuff_roll`（8 分）、`color_echo`（7 分）

**沙漏（肩胯同宽，腰明显收窄）**
- 核心挑战：身形已经很好，主要维护
- 策略：强调腰线（但不过度）、自然廓形、质感提升
- 避免：过度宽松掩盖曲线、过度紧身过于性感
- 最适合的技巧：`french_tuck`（8 分）、`belt_color_echo`（8 分）

### 6.2 身体特征修饰

除了五大类型，个体特征也需要针对性处理：

**长躯干**（腰线以下偏长）→ 核心策略：上移腰线。相关 modifier：`long_torso: +2 bonus` 给塞法系列
**短腿**（腿身比偏小）→ 核心策略：同色延伸 + 高腰 + 露踝。相关 modifier：`short_legs: +2 bonus` 给裤脚/塞法系列
**粗腰**（腰腹突出）→ 核心策略：半塞优于全塞、V 领优于圆领。相关 modifier：`thick_waist: caution` 全塞可能显腰粗
**宽肩**→ 核心策略：柔化肩线、V 领。相关 modifier：`broad_shoulders: +2 bonus` 给 V 领/解扣技巧
**窄肩**→ 核心策略：增加上半身体积。相关 modifier：`narrow_shoulders: +2 bonus` 给叠穿/层次技巧

### 6.3 露肤分寸

露肤是「视觉减重」的最高效手段。

**三大露肤位**：
1. **露踝**（脚踝是最细处）：露踝 = 腿到此最细处结束 → 利落显瘦
2. **露腕**（手腕是最细处）：露腕 = 手臂利落 → 精神感
3. **露锁骨/脖颈**（V 领延伸）：V 领 = 脖子+脸型纵向延伸 → 修长

**露肤的「不露」时机**：
- 正式场合（打领带/西装）：领口不露、脚踝不露
- 寒冷天气：用同色延伸代替露肤（如深色连裤袜 + 深色鞋）
- 户外活动：保护皮肤

**相关技巧**：`cuff_roll`, `sleeve_fold`, `button_rules`, `push_elbow_stack`, `roll_to_elbow`

## 七、场合 (Occasion)

穿搭从来不是在真空中存在的——它服务于一个具体的时间和地点。

### 7.1 正式度光谱

```
极休闲 ←――――――――――――――――――――→ 极正式
  沙滩  周末  日常  商务  半正式  正式  礼服
  短裤  牛仔裤  Polo  衬衫  Blazer  西装  Black Tie
```

**正式度的三个信号**：
1. **领口高度**：越低越休闲（T恤圆领 → Polo翻领 → 衬衫立领 → 西装驳领）
2. **扣子数量**：扣得越多越正式（0 颗敞开 → 1 颗解 → 全扣）
3. **面料挺括度**：越挺括越正式（亚麻 → 棉 → 羊毛 → 丝绸）

### 7.2 场合转换策略

**从日常→商务**：不换衣服，只改穿法
- 扣上最上面那颗扣 → 正式度 +20%
- 全塞替代半塞 → 正式度 +15%
- 加腰带（即使裤子不需要）→ 正式度 +10%

**从商务→日常**：
- 解开两颗扣 + 卷袖口 → 正式度 -30%
- 半塞替代全塞 → 正式度 -15%
- 不穿腰带 → 正式度 -10%

**相关技巧**：`button_rules`, `french_tuck`, `full_tuck`, `no_belt_styling`, `belt_casual_drape`
