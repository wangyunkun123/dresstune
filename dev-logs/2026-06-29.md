## 踩坑记录

### 坑 1：Python 3.9 类型注解
- **现象**：`X | None` 语法报 SyntaxError
- **原因**：macOS 默认 Python 3.9，PEP 604 在 Python 3.10+ 才原生支持
- **修复**：每个文件顶部加 `from __future__ import annotations`

### 坑 2：Seedream API 参数名错误
- **现象**：404 / 参数无效
- **原因**：初始代码用了 `"reference_image"` 和 `"mask_image"`，实际 API 参数是 `"image"`（数组）
- **修复**：对照原项目 `tools/generate.py` 完全重写 `generate_image()`

### 坑 3：火山引擎 Agent Plan 模型 ID 不匹配
- **现象**：`doubao-seed-1.8` 和 `doubao-seed-1.8-vision` 返回 404
- **原因**：Agent Plan 端点没有这些模型，模型列表与普通火山引擎不同
- **修复**：查询 Agent Plan 实际模型列表，全部更新为正确 ID
